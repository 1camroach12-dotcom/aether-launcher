# Aether Launcher

A high-performance Minecraft Bedrock client with enhanced UI, built-in mods, and auto-installation features.

## Features

- 🎨 **Custom UI/Menus** - Beautiful, intuitive interface
- ⚡ **Performance Optimized** - Enhanced game performance
- 🧩 **Mod System** - Easy mod management and installation
- 🚀 **Auto-Install** - Automatic mod installation on launch
- 🔧 **Extensible** - Build your own mods and features

## Project Structure

```
aether-launcher/
├── src/
│   ├── core/              # Core client logic
│   ├── ui/                # UI/Menu system
│   ├── mods/              # Mod loader and manager
│   ├── performance/       # Performance optimization
│   └── main.lua           # Entry point
├── mods/                  # Built-in mods
├── config/                # Configuration files
├── docs/                  # Documentation
└── README.md
```

## Getting Started

1. Clone this repository
2. Read [SETUP.md](docs/SETUP.md) for installation instructions
3. Check out [MOD_DEVELOPMENT.md](docs/MOD_DEVELOPMENT.md) to create your first mod
